# workshop-struts #
=================

TomcatでStruts開発を行うサンプルアプリケーション。

#### 概要
    TomcatでStruts開発を行う。

    - Actionにロジックを記述する。
    - Formで画面とActionのデータをやりとりする。
    - サーブレットはStrutsサーブレットを指定する。
    - JSPにカスタムタグでロジックを記述する。
    - JSPにEL式を記述する。
    - struts-confing.xmlに画面遷移を記述する。
    - ユニットテストの記述が大変。
