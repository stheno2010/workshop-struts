# workshop-struts #
=================

TomcatでStruts開発を行うサンプルアプリケーション。

### 1. 【一昔前のWebアプリ開発】 (12/20)
    IT業界の潮流と開発の移り変わりを紹介する。
    Spring/Junitを用いた開発を実演する。

#### 概要
    TomcatでStruts開発を行う。

    - Actionにロジックを記述する。
    - Formで画面とActionのデータをやりとりする。
    - サーブレットはStrutsサーブレットを指定する。
    - JSPにカスタムタグでロジックを記述する。
    - JSPにEL式を記述する。
    - struts-confing.xmlに画面遷移を記述する。
    - ユニットテストの記述が大変。
